# A little repo for my dotfiles UwU

Hiiiiiii !!!
Here's a little repository for all my dotfiles and my automation to replicate
as well as possible my home setup.

## Repository structure

```txt

repo
  |- doc
  |- dotfiles
  |- scripts
  |- fresh_install.sh
  |- README.md
  |- CREDITS.md

```

`./doc` contains all the necessary documentation to understand my code

`./dotfiles` contains all the configurations. Each folder will individually be
copied to `~/.config/<package-name>/`

`./scripts` contains all the scripts I use daily, and the scripts used to
automate my arch instance. The installation scripts won't be copied,
however, the other scripts will be to `~/scripts/`

## Automation

> [!WARNING]  
> The `fresh_install.sh` file is NOT intended for an OS that was already setup
> ! Run this ONLY if you have a fresh install. If you use it on an already
> setup system and it breaks your system, I cannot help you.

> [!NOTE]  
> There are multiple branches to this repository. Each are named in accordance
> to the type of hardware used.
> laptop is for my laptop
> tower is for my tower at home
> server is for my servers

Just after login, you can clone this repository and run `./fresh_install.sh`.
This will install everything the same way I have it and will try to reproduce
as best as possible my setup.
